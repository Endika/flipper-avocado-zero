# Avocado Zero

Tamagotchi-style avocado pit in a glass of water on Flipper Zero.

**Play:** water gets dirty over time. Clean when there is grime to grow roots. Too much grime ends the run. Fully grown roots unlock a short victory moment.

**Build:** `make test` (host unit tests), `make fap` (needs a local `flipperzero-firmware` tree and `./fbt`).
